# URL Shortener Demo

Rails web app that uses the [Url Shortener Gem](https://github.com/ahmadabdelhalim/url_shortener) to shorten urls. 

### Requirements

* [Rails 6] 
* [Ruby 2.6.5]

### How to use

```sh
$ git clone git@github.com:ahmadabdelhalim/url_shortener_demo.git
$ cd url_shortener_demo/
$ bin/setup
$ yarn install
$ rails s
```
